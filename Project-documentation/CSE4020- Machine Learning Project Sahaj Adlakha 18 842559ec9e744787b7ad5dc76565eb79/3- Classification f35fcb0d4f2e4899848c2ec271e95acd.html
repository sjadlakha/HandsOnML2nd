Stochastic Gradient Descent:

  • good for large dataset

  • Relies on randomness during training

Metrics for SGD:

    ‚Üí Accuracy above 95%!!! (fishy)

2. Confusion Matrix

3. Precision and Recall

‚ÜíEven precise metric

‚Üí F1 score = H.M of precision and recall (H.M. because it gives much more weights to low values)

Precision/ Recall Trade-off
  1. In applications, we need to consider precision (Restricted videos on Youtube) ‚áí less F.P.
  1. In other cases we need to consider recall (Detecting disease in a patient) ‚áí less F.N.
  • You can‚Äôt have both precision and recall high. If you increase precision, it will reduce recall, and vice versa.

‚áí "Generally" if you increase the threshold, precision will increase
